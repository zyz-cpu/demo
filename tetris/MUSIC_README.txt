# 俄罗斯方块 - 音乐添加说明

## 如何添加背景音乐

由于版权原因，游戏不自带音乐文件。请按以下步骤添加你自己的音乐：

### 步骤 1：准备音乐文件
将你喜欢的音乐文件（如 "We Don't Talk Anytime"）转换为以下格式之一：
- MP3 格式（推荐）：`bgm.mp3`
- WAV 格式（备用）：`bgm.wav`

### 步骤 2：放置音乐文件
将音乐文件复制到 `tetris` 文件夹中，与 `index.html` 同一目录。

### 步骤 3：运行游戏
打开 `index.html` 文件，点击"🎵 播放音乐"按钮即可开始播放背景音乐。

## 支持的音乐格式
- MP3 (.mp3) - 推荐
- WAV (.wav) - 备用

## 注意事项
- 音乐文件命名必须为 `bgm.mp3` 或 `bgm.wav`
- 如果浏览器提示需要用户交互才能播放音频，请先点击页面任意位置，然后再点击播放按钮
- 某些浏览器可能需要用户首次点击页面后才能自动播放音频
